# Who Ordered This?!

**Who Ordered This?!** is a browser-based diner game where players must figure out who is real, who is an imposter, and whether the order is actually on the menu.
file:///C:/Users/jnkam/OneDrive/Documents/diner-game/index.html 

Players must:
- **Serve** real customers with valid menu items
- **Reject** imposters
- **Reject** fake menu items
- Survive all **5 shifts** without making too many mistakes

---

## How to Play

Each round, a customer appears and asks for an item.

Your job is to decide whether to:

- **Serve** them
- **Reject** them

### Watch out for:
- suspicious names
- characters that look off
- menu items that are not actually on the diner menu

You only get **3 mistakes** before the game ends.

---

## Shift System

The game has **5 shifts**.  
To complete a shift, you must reach the required number of **correct orders**.

- **Shift 1** = 10 correct
- **Shift 2** = 20 correct
- **Shift 3** = 30 correct
- **Shift 4** = 40 correct
- **Shift 5** = 50 correct

Complete all 5 shifts to win.

---

## Features

- Title screen with animated game title
- Instructions screen with hints
- 5-shift progression system
- Real and fake customers
- Real and fake menu items
- Collapsible diner menu for checking valid foods
- Food handoff animation when serving
- Exit button to return to title screen
- Game over screen
- Win condition after finishing all shifts
